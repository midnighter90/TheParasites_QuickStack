# The Parasites QuickStack v1.0.0

Initial public portable release.

## Download

Use the attached release ZIP:

```text
TheParasites_QuickStack_v1.0.0.zip
```

## Installation

1. Close The Parasites.
2. Back up your saves.
3. Extract the ZIP.
4. Run `Install_QuickStack.cmd`.
5. Start the game.
6. Open a container.
7. Press `Ctrl+F9`.

## Hotkey

```text
Ctrl+F9
```

Moves matching inventory stacks into the currently opened container if that
container already contains the same item type.

## Target Version

Prepared for:

```text
The Parasites TP_Alpha_v_0.1.5.0.0
```

No compatibility is promised for other game versions.

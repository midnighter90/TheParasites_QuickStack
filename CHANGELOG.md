## v1.0.0 - 2026-05-24

Initial public portable release.

Added:

- `Ctrl+F9` QuickStack hotkey.
- Move matching inventory stacks to the currently opened container.
- Container matching based on item types already present in the opened
  container.
- Equipment/dedicated-slot skip logic for safer inventory handling.
- Anchored target-container handling during multi-item moves.
- Portable log path next to the installed mod.
- Complete UE4SS runtime payload for copy-only installation.
- Installer and uninstaller scripts with backup handling.
- Copyright, terms, installation, and third-party notice files.
